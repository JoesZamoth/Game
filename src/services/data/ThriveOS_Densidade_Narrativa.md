## 1. ANÁLISE DA DENSIDADE ATUAL

### 1.1 O Que o Jogo Já Tem ✅

```
FUNDAÇÃO SÓLIDA:
✅ Interface diegética completa
✅ Sistema de bateria narrativo
✅ Apps funcionais e temáticos
✅ Estrutura de 3 atos/10 capítulos
✅ Múltiplas interpretações de pistas
✅ Final aberto
```

### 1.2 O Que Falta Para Densidade Real ⚠️

```
LACUNAS DE PROFUNDIDADE:
⚠️ Conexões entre NPCs muito lineares
⚠️ Pistas ainda muito "solucionáveis"
⚠️ Falta de camadas ocultas nos diálogos
⚠️ Sistema de consequências limitado
⚠️ Pouca simulação de vida dos NPCs
⚠️ Ausência de meta-narrativa
⚠️ Segredos insuficientes por m²
```

### 1.3 Referências de Densidade

| Jogo | Densidade | O Que Aprender |
|------|-----------|----------------|
| **Disco Elysium** | 10/10 | Camadas ideológicas, NPCs vivos, stats internos |
| **Outer Wilds** | 10/10 | Conhecimento é progressão, tudo conectado |
| **Return of the Obra Dinn** | 9/10 | Dedução pura, sem mão na roda |
| **Her Story** | 8/10 | Fragmentação, ordem não-linear |
| **Pentiment** | 9/10 | Consequências invisíveis, ambiguidade moral |
| **The Case of the Golden Idol** | 8/10 | Puzzles contextuais, nomes/eventos/objetos |

## 2. CAMADAS DE PROFUNDIDADE NARRATIVA

### 2.1 Modelo de Camadas (Estilo Cebola)

```
┌─────────────────────────────────────────┐
│ CAMADA 1: SUPERFÍCIE                    │ <- 80% dos jogadores veem
│ Clara desapareceu, quem é o culpado?    │
├─────────────────────────────────────────┤
│ CAMADA 2: RELAÇÕES                      │ <- 50% descobrem
│ Triângulo amoroso, ciúmes, traição      │
├─────────────────────────────────────────┤
│ CAMADA 3: SEGREDOS DA CIDADE            │ <- 30% conectam
│ Escândalo antigo, famílias rivais       │
├─────────────────────────────────────────┤
│ CAMADA 4: TRAUMA DE EDUARDO             │ <- 20% percebem
│ Conexão pessoal, culpa do passado       │
├─────────────────────────────────────────┤
│ CAMADA 5: META-MISTÉRIO                 │ <- 10% desvendam
│ Investigação é parte de algo maior      │
├─────────────────────────────────────────┤
│ CAMADA 6: VERDADE FILOSÓFICA            │ <- 5% interpretam
│ Natureza da memória, identidade, verdade│
└─────────────────────────────────────────┘
```

### 2.2 Como Implementar Camadas

#### Camada 1: Plot Superficial (Obrigatório)
```
MISTÉRIO PRINCIPAL:
- Clara desapareceu em 23/04
- Última vista às 14h
- Casa revirada
- Celular apagado

OBJETIVO CLARO:
"Descubra o que aconteceu com Clara"

IMPLEMENTAÇÃO:
- Tutorial aponta para isso
- Todos os NPCs mencionam
- Checkpoints seguem essa linha
```

#### Camada 2: Rede de Relações (50% descobrem)
```
ESTRUTURA DE RELAÇÕES:

Clara ──(ex)── Marcos ──(rival)── Pedro
  │                                  │
(amiga)                          (irmão)
  │                                  │
 Ana ────────(caso)──────────────  Julia

SEGREDOS:
- Ana tinha caso com Pedro
- Marcos ainda ama Clara
- Julia inveja Clara
- Pedro deve dinheiro a Marcos
```

**Como Revelar:**
- Mensagens antigas descriptografadas
- Fotos em segundo plano
- Contradições em depoimentos
- Metadados de localização

#### Camada 3: História da Cidade (30% conectam)
```
EVENTO DO PASSADO (15 anos atrás):
- Acidente na fábrica local
- Pai de Clara morreu
- Pai de Marcos era dono da fábrica
- Encoberto como "acidente trabalhista"
- Clara descobriu recentemente a verdade

CONEXÕES:
- Clara confrontou Marcos
- Marcos tem medo de escândalo público
- Ana sabe do segredo (trabalhou na fábrica)
- Pedro era criança, mas viu algo
```

**Como Revelar:**
- Jornal antigo em arquivo descriptografado
- Depoimento de NPC secundário (velha vizinha)
- Foto antiga no álbum de Clara
- Gravação de confronto entre Clara e Marcos

#### Camada 4: Trauma de Eduardo (20% percebem)
```
CONEXÃO PESSOAL:
- Eduardo tem flashbacks durante investigação
- Ele também perdeu alguém em circunstância similar
- Pode estar projetando seu próprio luto em Clara
- Algumas "memórias" de Clara são dele mesmo
- Final questiona: Eduardo é narrador confiável?

SINTOMAS:
- Algumas deduções não fazem sentido
- NPCs questionam suas perguntas estranhas
- Bateria drena mais rápido em momentos emocionais
- Gravador capta sussurros que ele não percebe
```

**Como Revelar:**
- Reflexões ambíguas ("Isso me lembra de...")
- Foto antiga no celular dele (precisa procurar)
- NPC diz: "Você parece perturbado, detetive"
- Sistema de Dedução gera conclusões contraditórias

#### Camada 5: Meta-Mistério (10% desvendam)
```
TWIST META-NARRATIVO:
- Eduardo foi contratado por pessoa anônima
- Essa pessoa QUER que ele descubra a verdade...
- ...mas também está plantando pistas falsas
- Alguém está observando a investigação pelo ThriveOS
- Algumas mensagens são interceptadas/alteradas

EVIDÊNCIAS OCULTAS:
- Logs de sistema mostram acesso não autorizado
- Mensagens chegam antes de serem enviadas
- Arquivos "corrompidos" contêm código estranho
- IP de conexão muda sozinho
```

**Como Revelar:**
- App de "Logs do Sistema" desbloqueado no Cap. 8
- Decrypt revela backdoor no próprio ThriveOS
- NPC diz nunca ter enviado certa mensagem
- Jogador percebe que está sendo manipulado

#### Camada 6: Verdade Filosófica (5% interpretam)
```
TEMA PROFUNDO:
"A verdade é construída, não descoberta"

QUESTÕES:
- Toda investigação é narrativa
- Detetive escolhe qual história contar
- Evidências são interpretação, não fato
- Memória é ficção que acreditamos
- Clara pode ter desaparecido... ou nunca existido como pensamos

FINAL ABERTO:
Jogador escolhe qual "verdade" aceitar, sabendo que todas são parciais
```

### 2.3 Checklist de Densidade Narrativa

```
Para CADA capítulo, garanta:

☐ 3+ interpretações da cena principal
☐ 2+ segredos de NPC revelados parcialmente
☐ 1 conexão com evento passado
☐ 1 pista sobre trauma de Eduardo
☐ 1 elemento meta (glitch, código, anomalia)
☐ 5+ detalhes ambientais que contam micro-histórias
☐ 2+ evidências que contradizem algo anterior
☐ 1 escolha com consequência invisível (só aparece depois)
```

## 3. SISTEMAS EMERGENTES

### 3.1 Sistema de Confiança com NPCs

#### Mecânica Invisível
```javascript
cada_npc = {
  confianca: 0-100,  // invisível para jogador
  medo: 0-100,
  culpa: 0-100,
  suspeita_de_voce: 0-100
}

// Suas escolhas afetam:
if (pergunta_agressiva) {
  npc.confianca -= 15
  npc.medo += 10
  npc.defensividade += 20
}

if (confianca < 30) {
  npc.esconde_pistas = true
  npc.mente_mais = true
}

if (confianca > 70) {
  npc.revela_segredo_pessoal = true
  npc.ajuda_ativamente = true
}
```

#### Consequências Visíveis
```
BAIXA CONFIANÇA:
- NPC para de responder mensagens
- Depoimentos ficam vagos
- Mente sobre localização
- Apaga mensagens antigas
- Bloqueia você

ALTA CONFIANÇA:
- Envia mensagens espontâneas
- Compartilha fotos antigas
- Revela segredos de outros NPCs
- Oferece álibi verdadeiro
- Liga tarde da noite para confessar algo
```

#### Interface Oculta (Opcional)
Jogador nunca vê números, mas pode *sentir*:
- Tempo de resposta aumenta/diminui
- Tom de mensagens muda
- NPCs recomendam ou denunciam outros

### 3.2 Sistema de Rumores

#### Rede de Fofoca
```
Ana sabe sobre:
├─ Caso de Pedro com Julia
├─ Dívida de Marcos
└─ Segredo de Clara (parcial)

Marcos sabe sobre:
├─ Segredo de Ana
├─ Passado de Eduardo (!)
└─ Localização de Clara (?)

Pedro sabe sobre:
├─ Acidente na fábrica
├─ Relação Ana + Julia
└─ Medo de Marcos
```

#### Propagação Dinâmica
```
SE você revelar para Ana que sabe do caso dela:
  → Ana fica defensiva
  → Ana conta para Pedro que você sabe
  → Pedro para de responder por 1 dia
  → Pedro eventualmente confronta você
  → Novo diálogo desbloqueia
```

#### Implementação
```
SISTEMA DE EVENTOS:

1. Jogador pergunta a Ana sobre Pedro
2. Ana menciona "algo estranho"
3. Se jogador pressiona (escolha):
   → Ana revela caso
   → Confiança Ana -20
   → Flag: [ana_revelou_caso] = true

4. Próxima vez que falar com Pedro:
   IF [ana_revelou_caso] = true:
     Pedro: "Ouvi que você está fazendo
            perguntas sobre mim e Ana..."

5. Escolha:
   a) Negar → Pedro não acredita, bloqueia
   b) Confirmar → Pedro irritado mas honesto
   c) Desviar → Pedro desconfia
```

### 3.3 Sistema de Tempo e Urgência

#### Linha do Tempo Viva
```
DIAS DESDE DESAPARECIMENTO:
Dia 1-2: Pistas frescas, NPCs cooperativos
Dia 3-4: Evidências começam a desaparecer
Dia 5-6: NPCs nervosos, mudam histórias
Dia 7+: Caso esfria, pressão aumenta

EVENTOS TEMPORAIS:
- Mensagens chegam em horários específicos
- NPCs dormem (não respondem 23h-7h)
- Alguns locais só acessíveis em certos dias
- Clima muda (afeta fotos, atmosfera)
```

#### Pressão Temporal (Opcional)
```
SISTEMA DE URGÊNCIA SUAVE:

Não é time limit rígido, mas:
- Chefe envia mensagens cobrando progresso
- Mídia começa a noticiar caso
- Família de Clara pressiona
- Alguns NPCs saem da cidade
- Evidências são alteradas/destruídas

Jogador pode continuar investigando, mas sente peso
```

### 3.4 Sistema de Memória Falha

#### Depoimentos Mudam
```
PRIMEIRA VEZ (Dia 2):
Ana: "Vi Clara sair às 14h, sozinha"

SEGUNDA VEZ (Dia 5):
Ana: "Espera, acho que era 15h... ou 14h30?"

TERCEIRA VEZ (Dia 7):
Ana: "Tenho certeza que era 14h. Por que você
      continua perguntando isso?"

MECÂNICA:
- NPCs não têm certeza absoluta
- Memória é construída, não recuperada
- Jogador questiona: ela mente ou esqueceu?
```

#### Gravador Detecta Mudanças
```
FEATURE DO GRAVADOR:

"⚠️ Contradição Detectada"

Ana, 23/04 às 16h:
"Vi Clara sair às 14h"

Ana, 26/04 às 11h:
"Vi Clara sair às 15h"

[Confrontar Ana sobre isso?]
```

## 4. DENSIDADE DE PERSONAGENS

### 4.1 NPCs como Icebergs

#### Modelo de Profundidade
```
PARA CADA NPC:

10% VISÍVEL (primeiro contato):
- Nome, idade, ocupação
- Relação superficial com Clara
- Álibi básico

20% INVESTIGÁVEL (com perguntas certas):
- Relação real com Clara
- Segredo pessoal
- Motivação para mentir

30% ESCONDIDO (requer confiança):
- Trauma pessoal
- Conexão com evento do passado
- Segredo de família

40% OCULTO (requer multiple playthroughs):
- Papel no meta-mistério
- Verdadeira motivação
- Conexão com Eduardo
```

#### Exemplo: Ana Costa (Completa)

```
┌─────────────────────────────────────────┐
│ ANA COSTA - FICHA COMPLETA              │
├─────────────────────────────────────────┤
│                                         │
│ CAMADA 1: SUPERFÍCIE                    │
│ - Vizinha e melhor amiga de Clara       │
│ - 32 anos, professora                   │
│ - Última pessoa a ver Clara             │
│ - Álibi: estava em casa dando aula online│
│                                         │
│ CAMADA 2: INVESTIGÁVEL                  │
│ - Teve discussão com Clara semana antes │
│ - Clara descobriu caso dela com Pedro   │
│ - Tem medo que segredo vaze             │
│ - Mente sobre horário que viu Clara     │
│                                         │
│ CAMADA 3: ESCONDIDO                     │
│ - Mãe dela morreu no acidente da fábrica│
│ - Culpa família de Marcos               │
│ - Queria que Clara expusesse escândalo  │
│ - Mas tem medo das consequências        │

│ - Representa "testemunha não-confiável" │
│ - Espelha Eduardo (ambos buscam justiça)│

│ - Final dela muda baseado em escolhas   │
│                                         │

```

### 4.2 Diálogos em Camadas

#### Sistema de Perguntas Desbloqueáveis

```
CONVERSA COM ANA - PROGRESSÃO:

━━━ PRIMEIRO ENCONTRO (sempre disponível) ━━━
→ "Quando viu Clara pela última vez?"
→ "Como era a relação de vocês?"
→ "Sabe de alguém que quisesse mal a ela?"

━━━ APÓS ENCONTRAR FOTO (desbloqueia) ━━━
→ "Reconhece este homem na foto com Clara?"
→ "Clara mencionou estar com medo?"

━━━ APÓS DECRYPT CELULAR (desbloqueia) ━━━
→ "Por que Clara apagou suas mensagens?"
→ "Sobre o que vocês discutiram na semana passada?"

━━━ APÓS DESCOBRIR CASO (desbloqueia) ━━━
→ "Qual é sua relação com Pedro?" [!]
→ "Clara sabia sobre você e Pedro?"

━━━ APÓS ALTA CONFIANÇA (desbloqueia) ━━━
→ "Você está escondendo algo, não está?"
→ "Por que realmente me contratou?" [!!]
